# DIMMP Revendedor — App

## Como publicar (5 minutos, gratuito)

### Passo 1 — Criar conta no GitHub
1. Acesse https://github.com/signup
2. Crie uma conta gratuita (ex: usuário `dimmpcosmeticos`)

### Passo 2 — Criar repositório
1. Clique em "New repository"
2. Nome: `revendedor` (fica como `dimmpcosmeticos.github.io/revendedor`)
3. Marque "Public"
4. Clique "Create repository"

### Passo 3 — Subir os arquivos
Arraste os 5 arquivos para dentro do repositório:
- `index.html`
- `manifest.json`
- `sw.js`
- `icon-192.png`
- `icon-512.png`

### Passo 4 — Ativar GitHub Pages
1. Settings → Pages
2. Source: "Deploy from branch"
3. Branch: main → /root → Save

### Passo 5 — Pronto!
Link gerado: `https://dimmpcosmeticos.github.io/revendedor`

Mande esse link para o Bruno pelo WhatsApp.

## Como o Bruno instala no celular

**Android (Chrome):**
Menu ⋮ → "Adicionar à tela inicial" → Instalar

**iPhone (Safari):**
Botão compartilhar □↑ → "Adicionar à Tela de Início"

Vira ícone na tela, abre igual app, funciona offline, dados salvos no celular.
